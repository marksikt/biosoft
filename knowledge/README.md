## Convenție

Fiecare document conține:
1. **Principii arhitecturale** — cum se structurează un model al subsistemului
2. **Principii conceptuale per subsistem** — ce știm și cum se comportă
3. **Metodologie transferabilă** — instrumente și abordări reutilizabile
4. **Limitări cunoscute** — ce nu acoperă modelul sursă

Nivelul de detaliu: **conceptual** (nu parametric).
